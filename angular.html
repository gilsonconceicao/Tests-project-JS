O que é Angular JS ?

Angular é uma plataforma de aplicações web de código-fonte aberto e front-end baseado em TypeScript liderado pela Equipe Angular do Google e por uma comunidade de indivíduos e corporações.

Para que serve Angular ?

O Angular é um poderoso framework que utiliza HTML e TypeScript para criar a interface com o usuário, ou seja, o front-end em aplicações web, desktop e dispositivos móveis. A primeira versão foi lançada em 2010

É difícil aprender Anular?

Angular é um framework muito complexo. Para começar a utiliza-lo é preciso ter conhecimento sobre: WebPack; Configurar Módulos Node. js; NPM; e TypeScript que é relativamente fácil.
