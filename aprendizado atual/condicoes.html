PAGINA FEITA COM OPERADORES, FUNÇÕES, CONDIÇÕE E O DOM.

Operadores: Operadores em Javascript são símbolos especiais que envolvem um ou mais operandos com a finalidade de produzir um determinado resultado.

Funções: São um conjunto de código em bloco que quando é executado realiza uma determionada função.

Condições: Toma decisões no seu código, if e else (se e se não). Os tipos são: condições composta, aninhadas e switch.

DOM: Document Object Model, são modelos de objstos do documento. Onde pode acessar componentes de uma pagina HTML.
