Repetições em javascript

Repetições em javascript são basicamente testes lógicos de expressões com valores verdadeiro e falso. Com muitas opções de uso, tem a contagem de números, é uma ótima forma de práticar para quem estuda e aprender a utilzar.

Saiba mais

Exemplos das estruturas:

  • while (é realizada enquanto for verdadeira a condição)
  • do ... while (Feita com o teste lógico por último)

Afinal, o que é esse teste lógico?

Teste lógico é onde avalia a condição feita pela estrutura

while a lógica vem primeiro (while (c <= 5 )

Exemplo:
let c = 1;
while (c <= 5 ) {
console.log(` A contagem é ${c}`);
c++
}
resposta: 1 2 3 4 5

do ... while é o oposto, o teste fica por último.

Exemplo:
let c = 1;
do {
console.log(` A contagem é ${c}`);
c++
} while (c <= 5 );
resposta: 1 2 3 4 5
